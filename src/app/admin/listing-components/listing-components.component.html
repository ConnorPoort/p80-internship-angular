the following components can be used:
name: can be found at: customroute name:
event components/event event
traffic components/traffic event
weather components/weather weather
servers components/servers servers
financial components/financial financial
meeting components/meeting meeting
support components/support support
loop components/loop loop
news components/news news

Most components are only half a page high, Ex: weather and traffic. there even is one that is only a moving bar at the bottom of the page (news)

the single components are found at: host/components/name. and if you wish to use multiple components on one page you will need the custom route path: host/components/custom/componentname1-componentname2 as you can see you devide them by using '-' in between!
